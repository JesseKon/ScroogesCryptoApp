# Scrooge's Crypto App

## Installation
1. Install npm

2. Download backend packages
```
cd backend
npm install
```

3. Download frontend packages
```
cd frontend
npm install
```

## Running the app
1. Run the backend
```
cd backend
node app.js
```

2. Run the frontend
```
cd frontend
npm start
```

Backend will run at localhost:8000 and frontend at localhost:3000.
