# Scrooge's Crypto App
An app to inspect cryptocurrency prices in given time range.
![](./pic.png)

## Installation
1. Install npm

2. Install backend packages
```
cd backend
npm install
```

3. Install frontend packages
```
cd frontend
npm install
```

## Running the app
1. Run the backend
```
cd backend
node app.js
```

2. Run the frontend
```
cd frontend
npm start
```

Backend will run at localhost:8000 and frontend at localhost:3000.
